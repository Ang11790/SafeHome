# Backend SafeHome

Esta carpeta contiene el código backend (API Flask, utilidades de autenticación y Firestore) del proyecto SafeHome.

## Estructura sugerida
- app.py: Punto de entrada de la API Flask
- auth_utils.py: Funciones de autenticación y gestión de usuarios
- firebase_config.py: Configuración de Firebase y Firestore
- firestore_utils.py: Funciones para interactuar con Firestore
